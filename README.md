# Lista de Tareas CLI

Aplicación de línea de comandos creada en Python para gestionar una lista de tareas.

## Funcionalidades

- Agregar tareas.
- Ver tareas.
- Marcar tareas como completadas.
- Editar tareas.
- Eliminar tareas.
- Ver estadísticas.
- Guardar tareas en archivo JSON.

## Requisitos

- Python 3.8 o superior.

## Ejecución

```bash
python lista_tareas.py
